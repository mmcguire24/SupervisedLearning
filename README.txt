All of the code for running the models and generating the graphs is located in the python file named after each dataset


1.Download the code from https://github.com/mmcguire24/SupervisedLearning
2.Make sure you have both "pima.csv" and "sonar.csv" or download them from https://archive.ics.uci.edu/ml/datasets/Connectionist+Bench+(Sonar,+Mines+vs.+Rocks) and https://www.kaggle.com/uciml/pima-indians-diabetes-database
3. Make sure you have python 3 with Numpy,Keras,Scikit,Pandas, and Matplotlib
4.Run PimaClassificaiton.py and SonarClassification.py, All of the code for running the models and generating the graphs is located in the python file named after each dataset
